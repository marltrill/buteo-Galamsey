def type_check(variable: Any, types: list, name: str = '', allow_none=False, throw_error=True) ‑> bool

Utility function to type check the inputs of a function.

Args

variable : any
The variable to typecheck
types : tuple
A tuple of type classes. e.g. int, float, etc.

**kwargs: name (str): The name printed in the error string if an error is thrown.

allow_none (bool): Allow the type to be None

throw_error (bool): Should the function throw an error if the type
is wrong or return a boolean.

Returns

A boolean indicating if the type is valid. If throw_error an error is raised if the input is not a valid type.
